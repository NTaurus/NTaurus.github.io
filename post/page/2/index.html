博客搭建好啦！

十二月 20, 2019 发布在 文章

总结

感受

搭建的过程还是遇到挺多麻烦的，
毕竟很多东西都没有接触过。
从上周六开始着手准备搭建到搭建完成，
前前后后差不多一个星期，
因为还没结课，
白天的时间比较零散，
断断续续地研究搭建博客。
开始博客部署在github上，
访问实在是慢，
研究coding研究了好久，
coding这平台真是可（e）爱（xin），
上传文件夹显示 0 Byte。
无奈手动上传不了，
只能再学一下git的使用，
虽然跟着命令敲，
但不理解它什么意思，
真难受，
一个个ERROF瞬间懵，
心中无限疑惑，
催我上bing。
买一个域名还是挺有必要的，
github博客同步到coding，
用自己的域名双线解析，
可以加快访问博客的速度。
听说国内的域名要备案，
今天在DNSPod上解析域名，
确实要求要备案，
备案麻烦，
审核完得好多天。
今天加了播放音乐的小插件，
觉得博客有了灵魂，哈
博客总体就这样了，
等想起来或者偷学到什么技术的时候再弄。

需要注意的

hugo主题直接clone的话，
速度是十分慢的，
可以改电脑上的hosts文件。
文件路径： C:\Windows\System32\drivers\etc\hosts

git commit -m "XX"
双引号里面可以随便的写，但不能缺，
开始以为注释，原来也是属于命令的一部分。

SSL安全证书得每三个月重新申请，
重新申请的时候，
先停用github的海外解析再申请。

如果在git的CMD里验证coding或者github身份的时候，
不小心输入错误的密码或账号，
可以到电脑control->用户账户->windows凭证, 底下有个普通凭证,
把账号删掉再重新关联仓库就可以了（或者直接修改账号信息）。

我的博客开通啦

十二月 16, 2019 发布在 文章

做个小小的记录

好久之前就想开一个博客，
也曾在一些博客网站上注册过，
各式各样的广告让我眼花缭乱，
还有一堆限制（一脸嫌弃）。
后来看到有搭建个人博客的教程，
便有了自己动手搭建一个属于自己的博客的想法，
但要准备PAT乙级考试跟四级（不，你没有准备四级。。下次再来的哭脸.png）
所以就拖到了现在。

开始挺棘手的，
完全没有经验，
在不理解啥含义的情况下，
在CMD里跟着教程瞎敲命令（真是蠢得可以），
期间还卡了好多次，
还得感谢SAquarius呆神的不吝赐教，
在SAquarius的博客学了不少东西，再次感谢！

开博客是想在往后的刷题跟学习中做一些笔记。
要准备明年三月的：甲级考试、天梯赛跟蓝桥杯（甲级还是有点方，硬着头皮肝它）
所以近期可能是复习数据结构的知识还有刷题巩固。

除此之外，
也可能会记录自己的最近的生活状态，
就当这里是树洞吧。

都怪你老让我吃菜
不改host网速慢得让我目瞪口呆
多少次clone的失败是git不肯给我的爱
多少次pull的失败是gayhub在笑我活该
重来重来
冷冷的代码在我脑里胡乱的嗨
送它离开千里之外
我的blog终于怀上了胎
为养这个仔
硬着头皮跨过markdown的这障碍
搬砖搬到了现在
唉
